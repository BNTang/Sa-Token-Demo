## 默认值

```java
// 成员变量的默认值
public class DefaultValues {
    byte b;      // 0
    short s;     // 0
    int i;       // 0
    long l;      // 0L
    float f;     // 0.0f
    double d;    // 0.0d
    char c;      // '\u0000' (空字符)
    boolean bool;// false
}

// ⚠️ 局部变量没有默认值，必须初始化
public void method() {
    int x;
    // System.out.println(x);  // 编译错误!
}
```

## 字面量写法

```java
// 整数
int decimal = 100;       // 十进制
int binary = 0b1100100;  // 二进制 (Java 7+)
int octal = 0144;        // 八进制
int hex = 0x64;          // 十六进制
long big = 100L;         // long 类型加 L

// 可用下划线分隔 (Java 7+)
int million = 1_000_000;

// 浮点
float f = 3.14f;         // float 必须加 f
double d = 3.14;         // 默认是 double
double sci = 3.14e2;     // 科学计数法 = 314.0

// 字符
char c1 = 'A';
char c2 = 65;            // ASCII 码
char c3 = '\u0041';      // Unicode

// 布尔
boolean t = true;
boolean f = false;
```

## 类型转换

```
┌─────────────────────────────────────────────────────────────┐
│                    类型转换                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   自动转换 (小 → 大):                                        │
│   byte → short → int → long → float → double               │
│                    ↑                                        │
│                  char                                       │
│                                                             │
│   强制转换 (大 → 小): 可能丢失精度                          │
│   int i = 128;                                              │
│   byte b = (byte) i;  // -128 (溢出)                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

```java
// 自动类型转换 (隐式)
int i = 100;
long l = i;      // int → long 自动
double d = l;    // long → double 自动

// 强制类型转换 (显式)
double d = 3.99;
int i = (int) d;  // 3 (截断小数部分)

long l = 100L;
int i = (int) l;  // 100

// ⚠️ 溢出问题
int big = 130;
byte b = (byte) big;  // -126 (溢出!)
```

## 面试回答

### 30秒版本

> Java 有 **8 种基本数据类型**：4 种整数（byte/short/int/long）、2 种浮点（float/double）、1 种字符（char）、1 种布尔（boolean）。小类型可自动转大类型，反之需强制转换可能溢出。成员变量有默认值（数值 0，布尔 false），局部变量必须初始化。

### 1分钟版本

> **8 种基本类型**：
> - 整数：byte(1)、short(2)、int(4)、long(8)
> - 浮点：float(4)、double(8)
> - 字符：char(2)
> - 布尔：boolean
>
> **大小与范围**：
> - int：约 ±21 亿
> - long：加 L 后缀
> - float：加 f 后缀
>
> **类型转换**：
> - 自动：小→大
> - 强制：大→小，可能溢出
>
> **默认值**：
> - 成员变量有默认值
> - 局部变量必须初始化
